# 🎬 Live Cam Playlist Generator

Automatically fetches live HLS stream links from [chococams.com](https://chococams.com/model/) 
every 10 minutes and generates an M3U playlist.

## Features

- ✅ Scrapes **favourite models** by name
- ✅ Scrapes **top 10 online models** from the listing page
- ✅ Tries multiple stream sources (Stripchat, Chaturbate, BongaCams, etc.)
- ✅ Direct API calls for faster results
- ✅ Playwright browser fallback for JS-rendered pages
- ✅ Auto-updates every 10 minutes via GitHub Actions
- ✅ Generates standard M3U playlist with thumbnails and groups
